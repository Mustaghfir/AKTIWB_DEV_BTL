Unordered List

Unordered list is a random list with bullet points as mentioned below:

  • This is an unordered list.
  • We use 'ul' in 'html' to start Unordered List.
  • To mention list items we have to use 'li'.
  • It is mandatory to use opening "< >" and closing brackets. "< />"
    • for separate bullets in unordered list

Ordered List

Ordered List is the list with number bullets as mentioned below

  1. This is an Ordered List
  2. We use 'ol' in 'html' to start Ordered list
  3. To mention list items we have to use 'li' as mentioned earlier in 'unordered list
  4. It is mandatory to use opening and closing brackets as mentioned earlier in 'unordered list
